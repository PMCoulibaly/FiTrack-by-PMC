# 🔥 FitTrack — Guide d'installation sur iPhone

## Étape 1 — Créer un compte Vercel (gratuit)
1. Va sur **vercel.com**
2. Clique sur "Sign Up"
3. Connecte-toi avec GitHub (crée un compte GitHub gratuit si besoin sur github.com)

## Étape 2 — Uploader le projet
1. Va sur **vercel.com/new**
2. Clique sur **"Browse"** ou **"Import"**
3. Sélectionne le dossier `fittrack` (ce dossier)
4. Vercel détecte automatiquement que c'est un projet React (Create React App)
5. Laisse tous les paramètres par défaut
6. Clique **"Deploy"** → attends ~2 minutes

## Étape 3 — Obtenir ton lien
- Vercel te donne une URL du type : `fittrack-xxx.vercel.app`
- Teste-la dans Safari sur ton iPhone

## Étape 4 — Installer sur l'écran d'accueil iPhone
1. Ouvre l'URL dans **Safari** (important : pas Chrome !)
2. Appuie sur l'icône **Partager** (le carré avec une flèche vers le haut)
3. Fais défiler et appuie sur **"Sur l'écran d'accueil"**
4. Nomme l'app **FitTrack** et appuie sur **Ajouter**
5. ✅ FitTrack est maintenant sur ton écran d'accueil comme une vraie app !

---

## ✨ Fonctionnalités
- Configuration personnalisée (taille, poids, âge, objectif)
- Calcul automatique calories / macros / BMR / TDEE
- Journal alimentaire (22 aliments)
- Journal sportif (salle + domicile, 28 exercices)
- Programme hebdomadaire adapté
- Suivi du poids + projection semaine par semaine
- Hydratation
- Tout est sauvegardé localement sur ton iPhone

---

## 🆘 Problèmes fréquents
- **"Le site ne charge pas"** → Attends que le build Vercel soit terminé (barre verte)
- **"Pas d'option écran d'accueil"** → Utilise Safari, pas Chrome
- **"Les données disparaissent"** → Ne navigue pas en mode privé
